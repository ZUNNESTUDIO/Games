# 🎮 중앙초 친구들의 AI 게임 프로젝트

> 중앙초등학교 학생들이 ChatGPT와 대화하며 만든 게임 모음집

## 📖 프로젝트 소개

이 레포지토리는 중앙초등학교 학생들이 **AI와 협업**하여 제작한 게임들을 모아놓은 공간입니다.  
학생들은 메모장에 코드를 작성하고, 선생님과 함께 완성하여 실제 플레이 가능한 웹 게임으로 만들었습니다.

## 🎯 수업 목표

- 💬 **AI 도구 활용**: ChatGPT와 자연스러운 대화로 코딩하기
- 🧩 **논리적 사고**: 게임 로직 설계 및 문제 해결
- 🎨 **창의성**: 나만의 게임 아이디어 구현
- 👥 **디지털 리터러시**: 코드의 기본 구조 이해
