# 온유 - FPS ALL IN ONE 게임

## 게임 설명
Unity로 제작한 종합 FPS 게임입니다.

## 주요 기능
- 캐릭터 커스터마이징 (머리/피부/얼굴 선택)
- 플레이어 vs 적 AI 전투 시스템
- 2종류 무기 시스템 (총기 교체 가능)
- 점수 시스템 (플레이어/적 승수 기록)
- 승리/패배 판정
- 리스폰 시스템

## 사용 기술 
- Unity
- C#
- NavMesh AI

## 스크립트 적용 방법
- `FPS_ALLINONE_FULL.cs` 스크립트를 Unity 프로젝트의 `Scripts` 폴더에 추가한 뒤, 게임 매니저 오브젝트에 부착합니다.
- 인스펙터에서 플레이어/적 프리팹, 스폰 위치, UI(Text, 패널) 참조를 연결합니다.
- `HairColors`, `SkinColors`, `FaceTextures` 배열에 원하는 머리/피부/얼굴 텍스처와 머티리얼을 추가하면 커스터마이징이 활성화됩니다.
- 게임을 실행하면 플레이어와 적이 자동으로 스폰되고, 3선승제로 승리/패배 UI가 표시됩니다.
